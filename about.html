About Student Grade Calculator

The Student Grade Calculator is a simple web application developed using HTML, CSS, and JavaScript. It helps students calculate their total marks, average, percentage, grade, and pass/fail status quickly and accurately.

Project Objectives

  • Calculate total marks automatically.
  • Calculate average and percentage.
  • Assign grades based on percentage.
  • Display pass/fail status.
  • Provide an easy-to-use interface.

Features

✔ Fast Calculation

Instantly calculates marks and grades.

✔ Responsive Design

Works on desktops, tablets, and mobile phones.

✔ Easy Navigation

Simple menu for moving between pages.

✔ Print Result

Students can print their result page.

Technologies Used

Technology Purpose
HTML5 Structure of the website
CSS3 Styling and responsive design
JavaScript Grade calculation and data storage
